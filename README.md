# Surf — Task Manager PWA

Мобильный таск-менеджер со свайп-управлением на TinyGesture 3.0.  
Single-file PWA · Firebase Realtime Sync · Offline-first

## Навигация

### Верхняя панель (хедер)
- **День** — утренний/вечерний ритуал + задачи на сегодня (⭐)
- **Задачи** — основной рабочий экран с суб-табами
- **Заметки** — текстовый блокнот

### Меню (⋮)
- ♠ **Поток** — Tinder-карточки для разбора задач
- ▤ **Проекты** — просмотр по проектам со свайпом

### Суб-табы «Задачи»
| Таб | Описание |
|-----|----------|
| Список | Строки с drag-to-reorder |
| Карточки | Карточки с свайпом (→ готово, ← удалить) |
| Личные | Персональные задачи (не в Firebase) |
| ⭐ Сегодня | Задачи отмеченные на день |
| 🃏 | Быстрый вход в Поток |

## Жесты

| Жест | Где | Действие |
|------|-----|----------|
| Свайп → | Карточки, День | Выполнить |
| Свайп ← | Карточки, День | Удалить / убрать из дня |
| Свайп ←→ | Строка проекта (proj-bar) | Переключить проект |
| Тап + | Карточки (FAB) | Новая задача |
| Длинный тап + | Карточки (FAB) | Новый проект |
| Двойной тап лого | Хедер | Тёмная тема |
| Свайп от левого края | Везде | Открыть сайдбар |
| Длинный тап | Карточки потока | Сменить приоритет |
| Двойной тап | Карточки потока | Сменить проект |

### Поток (Tinder-карточки)
| Свайп | Действие |
|-------|----------|
| → вправо | ✓ Готово |
| ← влево | → В конец (потом) |
| ↑ вверх | 🗑 Удалить |
| ↓ вниз | ⭐ На сегодня |

**Фильтр по проектам** — чипсы вверху экрана Потока. Тап = вкл/выкл фильтр. Можно выбрать несколько проектов.

## Боковая панель проектов (гамбургер)
- Тап на проект — фильтрация задач
- Кнопки остаются видимыми до повторного нажатия на гамбургер
- Свайп влево — закрыть панель

## Firebase
- Realtime sync в shared-режиме
- Локальные комнаты с кодом
- Индикатор статуса (зелёный/красный/оранжевый)
- Push/Pull вручную через меню

## Файлы
```
index.html  — приложение (single-file, ~1020 строк)
sw.js       — Service Worker (network-first cache)
README.md   — документация
```

## Деплой
Выложить все 3 файла на GitHub Pages. Service Worker обновится автоматически (surf-v9).

## Технологии
- TinyGesture 3.0 (Apache-2.0, inlined)
- Firebase Realtime Database
- PWA + Service Worker (network-first)
- CSS Variables + Dark Mode
- Zero dependencies
